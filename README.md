# 车辆充电桩管理系统 java1007

## 项目概述

车辆充电桩管理系统是一款基于B/S架构和MVC模式的应用系统，采用前后端分离的设计模式，前端使用VUE框架，后端基于Java和SpringBoot技术。系统旨在为充电桩运营商提供一个高效、可靠的管理平台。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows/Mac
- **开发环境**:
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql 5.7+
- **技术栈**:
  - Java
  - Mysql
  - SSM（Spring、SpringMVC、MyBatis）
  - MyBatis-Plus
  - VUE
  - jQuery
  - HTML
- **数据库可视化工具**: Navicat
- **服务器**: SpringBoot内嵌Apache Tomcat

## 功能模块

系统包含以下角色：

- 管理员
- 维修员
- 前台

### 功能列表

1. **通用功能**
   - 登录功能
   - 注册功能

2. **管理端**
   - 首页
   - 个人中心
   - 维修员管理
   - 用户管理
   - 电桩类别管理
   - 充电桩管理
   - 充电桩报修管理
   - 维修回复管理
   - 系统管理

3. **维修员**
   - 系统首页
   - 个人中心
   - 充电桩报修管理
   - 维修回复管理

4. **前台**
   - 首页
   - 个人中心
   - 充电桩
   - 充电常识
